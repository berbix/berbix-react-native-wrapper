# react-native-berbix-sdk

React native berbix sdk

## Installation

```sh
npm install react-native-berbix-sdk
```

## Usage

```js
import BerbixSdk from "react-native-berbix-sdk";

// ...

const result = await BerbixSdk.multiply(3, 7);
```
